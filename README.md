# NTT_modeling_KTsuruta
NTT 株価予測モデル（自社株買い意思決定支援） 
＃＃目次
1.プロジェクト概要
2.本コードで検証できること
3.実行環境


1. プロジェクト概要（Overview）  本プロジェクトは、NTT 財務部による自社株買いの意思決定を支援することを目的として、 NTT の株価時系列データを用いた 株価上昇確率予測モデル を構築したものです。  単なる株価予測ではなく、  「いつ買うべきだったか」  「買わなかった判断は正しかったか」  「将来◯営業日後に株価が上昇する確率はどの程度か」  といった 実務判断に直結するアウトプットを重視しています。
2. 本コードで検証できること　このコードを用いることで、株価がその日から20営業日先までの間で上がる確率を測ることがで、自社株買いにおける、買い付けのタイミングとして適当であるかを判断することができる。

3.実行環境：Python 3 (Google colabo), 必要なライブラリ：pandas, numpy, matplotlib, japanize-matplotlib, scikit-learn, yfinance
